家庭菜園を楽しむ IoTデバイスとアプリ Vodafon grow

20代向け植物育成キット

Role

チームプロジェクト, UXデザイナー, リサーチ

Project duration

2020.1 to 2020.8


Project Goal

都市部に住む若者が野菜を育てられるようにすること。

Target Audience

20代の学生とエンジニア


Key challenges

植物を枯らさずに育てるツールを開発します。

Techniques
Adobe XD, Illustrator, Photoshop, Arduino, Fusion360

About the project

SMART GROW KIT II

OVERVIEW

Amsterdam University of Applied Science's project/アムルテルダム応用科学大学のプロジェクト

コミュニケーションがとれない植物の健康状態を知ることができるツールを作ります。

  • インタビューと観察を行い、植物との関わり方を聞きました。
  • プロトタイプの制作と改善
  • ユーザーテスト

ユーザーリサーチについて

観察とインタビューを実施:

プランターと植物の種を用意して20代の協力者に育ててもらい、インタビューを実施しました。


水やりが分からない:

水をどれくらいあげれば良いのか分からない。


植物を枯らしてしまう:

学校が忙しいので、気づいたら枯らしてしまう。


初期デザインのコンセプト:

シンプルに情報が伝わる画面、楽しく育てられる情報を提供する、壊れにくく信頼感があるデバイスを作ります。

ユーザーテストについて

手順:

作製したプロトタイプを用いて、ユーザの欲していることは何かを明らかにしていき、デザインをブラッシュアップしました。


ユーザが気になるプロトタイプのアイデアについて意見を聞きます。

紙のUIパーツを利用し、即席のワイヤーフレームを作ってもらいます。

欲しい機能をユーザに白紙ワイヤーフレームに描いてもらいます。

調査結果:

アプリで覚えることは少ない方が良い。

20代の利用者が飽きないで使い続けられるコンテンツを提供する。

植物の状態にすぐにアクセスできる。

利用者は植物をきれいに撮れないと感じていたことです。

モックアップについて

改善:

テスト協力者が気にした日光と水、気温の情報は目立たせます。植物の写真を撮るときにガイドを表示します。

モックアップの説明:

センサが取得した植物の状態 ( 日光、水分量、温度 ) をわかりやすく伝える方法と楽しんで育てるための コンテンツを検討しました。

植物が何を欲しているのか知らせるための状態を表示する画面を作りました。

育てることのモチベーション維持のため、成長の記録を残せる画面を作りました。

育てることのモチベーション維持のため、今まで育てた植物の一覧が見られる画面を作りました。

結論と学び

結論:

展示会の開催、Vordafoneへのプレゼンテーションを通して大学のインキュベータープログラムに取り上げてもらうことができました。

学び:

植物を育てるプロセスは植物を育てているユーザーの課題を知るのに役立ちました。
